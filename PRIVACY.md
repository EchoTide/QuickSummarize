# Privacy Policy

Last updated: 2026-03-18

QuickSummarize is a Chrome extension that helps users summarize webpages and ask questions about the current page inside Chrome Side Panel.

## Overview

QuickSummarize does not operate a QuickSummarize-owned relay server for user requests. The extension stores user settings locally in the browser and sends requests directly to the AI provider configured by the user.

## Data the Extension Handles

QuickSummarize may handle the following data only when required for its core functionality:

- User-provided configuration data, such as API base URL, model name, API key, and language preference
- Current webpage content, such as page title, URL, selected text, and readable page text
- AI responses returned by the user-configured provider

## How Data Is Used

QuickSummarize uses this data only to provide its core features:

- Summarizing the current webpage
- Answering questions about the current webpage
- Remembering local configuration and interface preferences

The extension does not use user data for advertising, profiling, or unrelated analytics.

## Local Storage

User settings, including API credentials entered by the user, are stored locally in the browser through Chrome extension storage so the user does not need to re-enter them each time.

## Third-Party Processing

When the user requests a summary or asks a question, relevant webpage content is sent directly to the AI service configured by the user. QuickSummarize does not proxy these requests through its own server.

Users are responsible for reviewing the privacy practices of the AI provider they choose to use.

## Data Sharing

QuickSummarize does not sell user data.

QuickSummarize does not share user data with advertisers or data brokers.

QuickSummarize only transmits data to the AI provider explicitly configured by the user for the purpose of generating summaries or answers.

## Data Retention

QuickSummarize does not maintain a QuickSummarize-owned backend database for user content or API credentials.

Data stored by the extension remains in the user's local browser storage unless the user changes or removes it.

## Security

QuickSummarize stores settings locally and relies on the browser's extension storage environment. However, no software system can guarantee absolute security, and users should protect access to their browser profile and any API credentials they enter.

## User Control

Users can control their data by:

- Editing or deleting their extension settings
- Removing the extension from Chrome
- Choosing which AI provider to use
- Deciding when to summarize a page or send page content for processing
